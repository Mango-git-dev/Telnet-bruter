# Telnet Bruter

### Installation
```sh
git clone https://github.com/Mango-git-dev/telnet-bruter.git
cd ./telnet-bruter/
gcc ./*.c -o telnet-bruter -pthread -std=c99 -fcommon
sh bruter.sh

```
